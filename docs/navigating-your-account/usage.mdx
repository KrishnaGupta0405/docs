The usage page shows your current account usage and message consumption history, helping you understand how you're using your ContextGPT subscription.

## Usage overview

View your current consumption across all plan limits:

### Messages

Track your message usage:

* **GPT-4.1 mini messages** - Standard model usage and remaining quota
* **GPT-4.1 messages** - High-quality model usage and remaining quota

### Chatbots

Monitor your chatbot count:

* **Active chatbots** - Currently deployed
* **Limit** - Maximum allowed on your plan

### Content

Track your training data:

* **Pages** - Total pages consumed and remaining quota (1 page = 2,500 cleaned characters)
  * Includes all content from web pages, files, documents, and other sources

### Team

View team member usage:

* **Active members** - Users with access
* **Limit** - Maximum team size on your plan

## Usage history

View your message consumption over the last 12 months with an interactive chart.

### Viewing usage history

1. Navigate to **Usage** page
2. Select model type using the toggle buttons:
   * **GPT-4.1-mini** - View standard model usage
   * **GPT-4.1** - View high-quality model usage
3. Review the bar chart showing monthly consumption
4. Compare usage against your quota (shown as a horizontal line)

The chart displays:

* Monthly message consumption for the selected model
* Your quota limit as a reference line
* Last 12 months of data

<Tip>Switch between model types to see how your usage is distributed across GPT-4.1 and GPT-4.1-mini.</Tip>

## API key

If you have API access enabled, your API key is displayed at the top of the usage page. Use this key to authenticate API requests to ContextGPT.
