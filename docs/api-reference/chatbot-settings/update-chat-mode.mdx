---
title: Update Chat Mode
description: Update the chat mode of a chatbot
sidebarTitle: Update Chat Mode
---

To update the chat mode of your chatbot, use the **Update Chatbot Settings** route (`PATCH /v1/chatbots/chatbot/{chatbotId}/settings`) and include the `chatMode` field in the request body.

The `chatMode` field is part of the general chatbot settings — there is no separate endpoint for it.
